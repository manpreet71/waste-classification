About WasteIQ

An AI-powered waste classification system built as a Final Year Project to promote smarter, more sustainable waste disposal practices.

Project Overview

WasteIQ is an end-to-end deep learning system that classifies waste images into 8 categories and provides detailed recycling and disposal suggestions. The system was built to address a real-world problem — most people do not know how to correctly classify and dispose of their waste, leading to contamination of recycling streams and unnecessary landfill waste.

The project combines a custom-trained EfficientNetB0 model, a FastAPI backend, and a responsive web frontend to deliver a complete, deployable waste management assistant.

How the Model Works

1
Image Input
User uploads a waste image via the web interface. The image is sent to the FastAPI backend as a multipart form upload.
2
Preprocessing
The image is resized to 224×224 pixels and converted to a NumPy array. EfficientNetB0's built-in preprocessing normalises pixel values.
3
Model Inference
The preprocessed image is passed through EfficientNetB0. The model outputs a probability distribution across 8 waste classes. The class with the highest probability is selected.
4
Metadata Lookup
The predicted class is used to fetch metadata — biodegradable status, reusability, recyclability, and a detailed disposal suggestion.
5
Result Display
The result is returned as JSON and rendered on the classify page with a confidence bar, color-coded badges, and disposal guidance.

Dataset

The model was trained on the Garbage Classification dataset from Kaggle, originally containing 12 waste classes. The classes were remapped and merged into 8 final categories. The textile class was capped at 1,500 images to prevent class imbalance. The final dataset was split into 70% training, 15% validation, and 15% test sets.
